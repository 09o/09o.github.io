这些日子以来（一）

发表于 2017-10-01 | 分类于 个人记载 | 浏览 次
字数统计: 769 | 阅读时长 ≈ 3

一周过来，发现自己进度还真够慢的。原本打算在两周时间学完二十章的内容，最后却只完成了一小部分。从决定了读英文原版资料以来，我发现这一决定还是蛮正确的。由于看中文翻译资料时，其中有些地方，会出现翻译不是很准确，要不就是翻译起来并不是很好理解，相对来说后者更多吧。然后虽然在阅读英文资料时，中间会有一些单词不认识，但其实并不是很影响理解这一部分内容。而且最后会发现整个过程中，精力会比看中文资料要更集中。这样一来，学起来是慢了些，但整个学习效果和看中文书是差不太多的。但是为了有更好的效果，所以还是得加把劲提高英语水平。

最近在B站上关注了一个UP主，她每天会更新自己的考研准备学习过程。虽然所更新的视频没有太大参考价值，却让人觉得这种坚持很厉害，可以用来鼓舞激励自己吧。昨天整个夜里都在做梦，我不确定做了有多少个。按我现在回忆起来，至少有三个吧。虽说做梦是正常的，但是还是蛮影响睡眠质量的。差不多每做一个梦，就要醒一次，虽然会立即再次进入睡眠。不知不觉中就过去一个月的时间了，感觉在九月中也没有获得太多的东西，倒是精神上感觉被折腾了一番。会发现，学了后面的忘了前面的。当做一个题时，因为忘记了以前接触过的一个公式，而陷入不解。面对一个题没有思路时，有些痛苦，责怪当事人的能力不足。不得不再补以前的一些内容，而这样，会耗费更多的时间，对完成计划又有一些影响。眼看着快接近deadline了，却还有很多没有完成，终归于还是时间的安排规划没有做好。希望在这方面能够做的更好吧。

这周的阅读计划做的也不是很好，本计划的读完两本书也是没有达标。但也并不会因此降低目标说，改为一本。通过之前的经历，一周三四本我觉得都是可以完成的，放低目标，只会更加懈怠。当看的书足够多了，相信可以不用一昧的的参考别人推荐的“热销书”了，可以根据自己的判断来决定一本书合适不合适。当然有些热销书还是很值得看的，特别是一些经典著作。

最近在做的：

  • 学数学 《概率论与数理统计》
  • 背单词
  • 读书 《追风筝的人》、《东京本屋》
  • Coding
